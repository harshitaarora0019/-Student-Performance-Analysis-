# -Student-Performance-Analysis-
This project analyzes student academic performance using Power BI to understand how attendance, gender, section, and house influence final marks and results. The dashboard highlights trends through interactive visualizations such as attendance vs marks, average marks by section and gender, student distribution, and pass/fail analysis.
## Overview
This project analyzes student academic performance using Power BI to understand the impact of attendance, gender, section, and house on final marks and results.

## Dashboard Insights
- Attendance vs Final Marks relationship
- Average marks by section and gender
- Student distribution by house
- Pass vs Fail analysis across sections

## Tools Used
- Power BI
- Microsoft Excel

## Key Learnings
- Practical understanding of mean (average)
- Data visualization and storytelling
- Basic analytical thinking using real-world style data
